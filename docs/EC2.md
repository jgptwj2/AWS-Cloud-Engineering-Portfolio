# Amazon EC2

## Overview

WhaleBot v5 was deployed to an Ubuntu EC2 instance hosted on AWS.

## Skills Demonstrated

- Launching and managing EC2 instances
- SSH connectivity
- Linux administration
- Application deployment
- Software updates
- Troubleshooting

## Lessons Learned

Deploying an application in the cloud requires more than writing code. Managing Linux, maintaining SSH access, organizing deployments, and troubleshooting connectivity are all important parts of running a cloud-hosted application.
