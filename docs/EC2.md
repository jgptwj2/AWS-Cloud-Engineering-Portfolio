# Amazon EC2

## Overview

WhaleBot v5 was deployed to an Ubuntu Linux Amazon EC2 instance hosted on AWS.

## Technologies Used

- Amazon EC2
- Ubuntu Linux
- Elastic IP
- Public IPv4 Addressing
- SSH
- PEM Key Authentication
- Security Groups

## Practical Experience

While deploying and maintaining WhaleBot v5 I worked with:

- Launching and managing EC2 instances
- Connecting securely using SSH and PEM keys
- Configuring Security Groups
- Working with Elastic IP addressing
- Managing Public IPv4 connectivity
- Selecting AWS Regions and Availability Zones
- Starting, stopping and rebooting instances
- Deploying and maintaining Python applications
- Troubleshooting deployment issues
